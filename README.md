# Financial-Expense-Manager
A simple financial expense manager with login, dashboard, expense tracking, and reporting features using HTML, CSS, and JavaScript.


# Expense Manager Web Application

# Overview

Expense Manager is a web-based application developed using **HTML, CSS, and JavaScript** that helps users track and manage their personal expenses.

The application allows users to log in, add expenses, delete incorrect entries, and view a report summarizing the total spending. The interface is designed with a clean and aesthetic layout to provide a simple and user-friendly experience.

# Features

* Login interface
* Dashboard showing total expenses
* Add and manage expense records
* Delete incorrect entries
* Expense report page displaying total spending
* Modern aesthetic user interface

# Technologies Used

* HTML – Structure of the web pages
* CSS – Styling and user interface design
* JavaScript – Application logic and Local Storage for saving data

1. Download or clone the repository.
